# svg-codec
USAGE：
```shell
python main.py [-h] [--simple] action infile outfile
```
|参数|说明|
|:-:|:--|
|`-h`|帮助|
|`--simple`|使用直接编码|
|`action`|编码(`encode`)或者解码(`decode`)|
|`infile`|输入文件|
|`outfile`|输出文件|
